# Assignment 5 Solutions

This directory contains solutions for the JavaScript assignment questions.

## Files Description

- **`q1.js`**: Filter unique array members using `Set`.
- **`q2.js`**: Filter anagrams using `Map`.
- **`q3.js`**: Implement inheritance up to 3 classes using ES6 `class` syntax (includes private and static members).
- **`q4.js`**: Implement inheritance and static functions using prototype-based approach.
- **`q5.js`**: Import a module (`geometry.js`) to calculate area of circle, rectangle, and cylinder.
- **`q6.js`**: Import a module (`uniqueArrayBuilder.js`) to filter unique elements in an array.
- **`q7.js`**: Flatten a nested array using `Array.prototype.flat()` and recursion.
- **`geometry.js`**: Module containing constants and functions for geometric calculations (used in `q5.js`).
- **`uniqueArrayBuilder.js`**: Module containing a function to filter unique array elements (used in `q6.js`).
